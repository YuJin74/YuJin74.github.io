安装deepin双系统

2020年04月13日 544 字 大概 2 分钟
  1. 1. 安装准备
  2. 2. 制作分区
  3. 3. 制作启动U盘
  4. 4. 安装deepin系统
  5. 5. 安装完成

出于学习Linux的心态, 为了防止自己有事没事打开steam,试着安装了一下deepin系统.

安装准备

  • 25G以上的空闲硬盘空间

  • 8G以上的U盘

制作分区

  1. 打开磁盘管理(开始菜单Windows 徽标上右键单击，选择磁盘管理)
  2. 选中一个有足够空间的分区
  3. 在所选分区单击右键,选择压缩卷
  4. 输入需要的空间,单位换算1GB=1024MB,我选择分配50G空间,所以就是输入50*1024=51200
  5. 压缩完成后就出现了未分配的分区
  6. 在该分区右键单击,选择新建简单卷,然后一路无脑下一步

制作启动U盘

  1. 在深度科技官方网站下载镜像文件和深度启动盘制作工具
  2. 校验下载的镜像文件的MD5值与下载页面提供的 MD5 值是否一致(MD5 值在立即下载按钮下方).打开命令提示符输入CertUtil -hashfile 文件路径 MD5来计算下载镜像的MD5值
  3. 插入你的U盘
  4. 打开下载好的 Deepin 安装镜像，双击运行 DEEPIN_B.EXE
  5. 点击上方的光盘图标选择 Deepin 的 ISO 安装镜像
  6. 点击下方的 U 盘图标选择要使用的 U 盘以制作 U 盘启动,勾选格式化时会出现提示，备份过数据后选择确定即可.再次点击要使用 U 盘即可确认选择，点击开始按钮开始制作

安装deepin系统

  1. 开机时进入BIOS界面(不同机型方法不同具体自行百度),将U盘设为首选启动项,保存并退出BIOS界面
  2. 等待重启即可进入deepin安装界面
  3. 然后跟着安装器提示一步步进行就行了 这个不用写吧?不用吧?不用吧?
  4. 等待安装完毕重启电脑即可进入deepin系统

安装完成
